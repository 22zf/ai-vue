# 个性化备考计划

根据用户的时间、基础、目标，生成定制化学习路线。

---

## 信息收集

### 必须收集

| 维度 | 问题 | 选项 |
|------|------|------|
| 岗位 | 目标岗位？ | 9大岗位 |
| 公司 | 目标公司？ | 8家特色公司 / 通用 |
| 阶段 | 求职阶段？ | 实习 / 校招 / 社招 |
| 时间 | 距面试还有多久？ | 1周 / 2周 / 1月 / 3月 / 半年 |

### 可选收集

| 维度 | 问题 | 用途 |
|------|------|------|
| 基础 | 哪些领域比较薄弱？ | 调整学习重点 |
| 项目 | 有没有项目？ | 决定是否需要项目包装指导 |
| 刷题 | LeetCode刷了多少？ | 决定手撕代码训练强度 |
| 经历 | 之前面试过吗？挂在哪里？ | 针对性补强 |

## 备考计划模板

### 1周速成计划

```
适用：面试在即，需要快速突击

Day 1-2: 八股文高频突击
- 只看 ⭐⭐⭐⭐⭐ 考点（约10-15个）
- 每个考点：理解核心要点 + 背回答框架
- 输出：高频考点清单 + 每个考点的1分钟回答

Day 3-4: 手撕代码速练
- 只练 ⭐⭐⭐ 及以下的通用高频题
- 重点：反转链表、二分查找、LRU缓存、两数之和
- 每题限时15分钟

Day 5: 项目包装
- 提炼项目亮点（3个）
- 准备项目叙述（1分钟版 + 3分钟版）
- 预判5个追问并准备回答

Day 6: 模拟面试
- 1轮完整模拟面试
- 记录薄弱点

Day 7: 查漏补缺
- 针对模拟面试暴露的问题重点复习
- 再过一遍高频考点
```

### 2周标准计划

```
适用：有一定基础，需要系统准备

Week 1: 知识体系搭建
Day 1-2: 八股文高频（⭐⭐⭐⭐⭐）系统学习
Day 3-4: 八股文中频（⭐⭐⭐⭐）补充
Day 5: 手撕代码基础题（⭐⭐以下）全部过一遍
Day 6-7: 手撕代码进阶题（⭐⭐⭐）练习

Week 2: 项目+模拟+补强
Day 8: 项目包装 + 追问准备
Day 9: 公司特色考点突击
Day 10-11: 模拟面试（2轮）
Day 12-13: 薄弱点专项突破
Day 14: 最终复习 + 心态调整
```

### 1月系统计划

```
适用：时间充裕，追求全面

Week 1: 基础知识
- 八股文全部考点系统学习
- 每天学1-2个知识领域
- 边学边做笔记

Week 2: 手撕代码
- 按岗位分类刷题
- 每天3-5题
- 重点题写2-3遍

Week 3: 项目+设计
- 项目包装和亮点提炼
- 系统设计题练习
- 场景题准备

Week 4: 模拟+冲刺
- 3轮模拟面试
- 薄弱点专项训练
- 高频考点最终复习
```

### 3月长期计划

```
适用：提前准备，稳扎稳打

Month 1: 基础夯实
- 八股文全部考点深入学习
- 每个考点做到能讲5分钟
- 手撕代码每天2题

Month 2: 项目+实战
- 做1-2个有亮点的项目
- 或优化现有项目
- LeetCode按专题刷

Month 3: 冲刺+模拟
- 模拟面试每周2次
- 薄弱点反复训练
- 面试前1周：高频考点速览
```

## 薄弱点诊断

### 诊断方式

```
1. 快速问答：每个知识领域出1题，根据回答评估掌握度
2. 用户自评：让用户对每个领域打分(1-5)
3. 模拟面试：通过模拟面试暴露薄弱点
```

### 掌握度评估标准

| 等级 | 描述 | 建议 |
|------|------|------|
| ⭐ | 只听过名词 | 从零学起，看教程+做笔记 |
| ⭐⭐ | 知道大概 | 系统学习，重点理解原理 |
| ⭐⭐⭐ | 能说出要点 | 深入细节，准备追问 |
| ⭐⭐⭐⭐ | 能展开讲解 | 练习表达，控制时间 |
| ⭐⭐⭐⭐⭐ | 能应对追问 | 定期复习，保持手感 |

### 诊断输出格式

```
## 📊 知识掌握度雷达图

| 领域 | 掌握度 | 优先级 |
|------|--------|--------|
| C++语言基础 | ⭐⭐⭐⭐ | 低 |
| 操作系统 | ⭐⭐ | 🔴高 |
| 计算机网络 | ⭐⭐⭐ | 🟡中 |
| 数据库 | ⭐⭐ | 🔴高 |
| 设计模式 | ⭐⭐⭐ | 🟡中 |

### 建议学习顺序
1. 🔴 操作系统（当前最弱且高频）
2. 🔴 数据库（当前最弱且高频）
3. 🟡 计算机网络（中等但高频）
4. 🟡 设计模式（中等且中频）
```

## 每日学习建议

### 工作日（2-3小时）

```
1h  八股文学习（1-2个考点，深入理解）
1h  手撕代码练习（1-2题，限时完成）
30m 复习昨天学的内容（间隔重复）
```

### 周末（5-6小时）

```
2h  八股文系统学习
2h  手撕代码专题训练
1h  项目包装/模拟面试
1h  复习总结
```

### 碎片时间

```
- 通勤：听技术播客/复习笔记
- 午休：刷1道LeetCode简单题
- 睡前：过一遍今天学的回答框架
```
