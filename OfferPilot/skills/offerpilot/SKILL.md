# OfferPilot — AI面试备考助手

## 技能描述

面试备考知识库，支持12个岗位的八股文生成、模拟面试、项目分析、备考计划和专项突破。

## 执行逻辑

当用户触发此技能时，按以下流程执行：

### Step 1: 识别需求

从用户输入中提取三个维度：

1. **岗位**（必选）：匹配岗位映射表
2. **公司**（可选）：匹配公司特色表
3. **阶段**（可选）：实习 / 校招 / 社招

### Step 2: 识别模式

| 用户输入 | 模式 |
|----------|------|
| 面试准备/八股文/面经 | 八股文生成 |
| 模拟面试/面试练习 | 模拟面试 |
| 项目路径/分析项目/我的项目 | 项目分析 |
| 备考计划/学习计划/怎么准备 | 备考计划 |
| 专项/薄弱/XX比较弱 | 专项突破 |

### Step 3: 读取知识库

读取项目目录下的对应文件：

**岗位映射：**

- C++后端 → references/cpp-backend.md
- Java后端 → references/java-backend.md
- 前端 → references/frontend.md
- AI算法 → references/ai-algorithm.md
- 测试开发 → references/test-dev.md
- 嵌入式 → references/embedded.md
- 移动端 → references/mobile.md
- 数据分析 → references/data-analysis.md
- 云计算运维 → references/cloud-ops.md
- AI Agent → references/ai-agent.md
- AIGC → references/aigc.md
- MLOps → references/mlops.md

同时读取 references/handwrite.md 和 references/project-packaging.md。

### Step 4: 按模式输出

**八股文生成：**
- 八股文清单（按频率排序，含核心要点+面试问法+回答框架+延伸考点）
- 手撕代码推荐（含难度+频率+解题思路+易错点）
- 项目包装建议
- 公司特色调整（若提及公司）

**模拟面试：**
- 逐题出题，用户回答后逐题点评
- 题目顺序：自我介绍→八股文→项目深挖→手撕代码→反问
- 结束后输出面试报告

**项目分析：**
- 读取项目代码→识别技术栈→分析结构→找亮点
- 输出：分析报告+3个亮点+5个追问预判+10题面试题

**备考计划：**
- 按时间选模板：1周速成/2周标准/1月系统/3月长期
- 输出：每日学习安排+重点分配

**专项突破：**
- 难度递进出题：L1概念→L2原理→L3应用→L4对比→L5设计
- 训练结束输出掌握度评估

## 公司特色

| 公司 | 手撕特点 | 八股重点 | 追问风格 |
|------|----------|----------|----------|
| 字节 | 题量大(2-3道)、难度高 | C++深挖、网络、算法 | 极细追问 |
| 腾讯 | 中等、偏基础 | 网络、OS、底层原理 | 重理解深度 |
| 阿里 | 偏业务场景 | Java八股、业务理解 | 重业务思考 |
| 百度 | 算法为主 | 算法、技术细节 | 重原理 |
| 美团 | 中等、偏实际 | Java、分布式 | 场景题多 |
| 快手 | C++底层题多 | C++、音视频 | 中等 |
| 小红书 | Java、高并发 | Java、高并发 | 重业务场景 |
| 华为 | 通信协议、嵌入式 | 通信、嵌入式 | 综合素质 |

未知公司：根据行业/规模/技术栈智能推断面试风格。

## 异常处理

- 岗位无法识别：列出12个岗位让用户选择
- 公司不在特色表：智能推断，标注"智能推断"
- 用户只说"面试准备"：依次询问岗位→公司(可选)→阶段(可选)
